# Angular quick start


#### Development server
	ng serve

http://localhost:4200/
	
#### Code scaffolding
	ng g component components/name

`directive | pipe | service | class | module | component`

#### Build
	ng build
	
generate in dist/
	
	ng build -prod
	
#### Unit tests
	ng test

[Karma](https://karma-runner.github.io)

#### e2e tests
	ng serve
	ng e2e
	
[Protractor](http://www.protractortest.org/)
